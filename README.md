## General info
This app allows people to register and customize the way they want to taste their coffees.
The app covers the following contents:

- Introduction
- Firebase Setup
- Basic App Structure
- Firebase Auth
- Anonymous Sign In
- Custom User Model
- Streams
- Provider Package
- Signing Out
